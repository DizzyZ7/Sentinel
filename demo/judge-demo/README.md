# Sentinel judge demo

Intentionally small, non-deployable fixtures for the built-in product tour:

- a confirmed SQL injection with a proof-passing parameterized patch;
- a deterministic candidate rejected as a false positive;
- a cosmetic SQL patch that applies but fails regression proof.

The default one-click tour uses an explicitly labelled deterministic review replay. Use the live demo button to send the same sanitized evidence to GPT-5.6.
